# Event Registration System

A web-based event registration system built with PHP, MySQL, and Bootstrap.

## Features

- User registration and login
- Event browsing and registration
- Admin dashboard for managing users and registrations
- Responsive design with Bootstrap

## Installation

1. Import the `database/event.sql` file into your MySQL database.
2. Update the database credentials in `php/db.php`.
3. Start your web server (e.g., XAMPP) and navigate to `index.html`.

## Usage

- Visit `index.html` to see the landing page.
- Register a new account or login.
- From the dashboard, browse and register for events.
- Admins can access `admin.html` to manage the system.

## Technologies Used

- HTML, CSS, JavaScript
- Bootstrap 5
- PHP with PDO
- MySQL
